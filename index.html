1. Consideraciones Generales

RespirApp es una plataforma gamificada diseñada para el entrenamiento respiratorio en población pediátrica mediante el uso de un sensor de flujo MBMED acoplado al dispositivo FluxMed GrH. El software integra módulos de adquisición de señal, juegos terapéuticos, gestión de usuarios y generación de reportes clínicos. Este manual describe las funciones principales para administradores, profesionales y pacientes, y provee enlaces a tutoriales en video que muestran el uso de cada componente.

1.1. Requisitos del Sistema

Sistema operativo: Windows 10 o superior.
Python 3.10+ instalado.
Sensor de flujo MBMED® conectado al equipo FluxMed GrH.
Accesorios necesarios: boquilla espirométrica, filtro HME, clip nasal y, de ser necesario, adaptadores 3D.
Conexión USB disponible para establecer comunicación serie.

1.2. Uso de Boquillas, Filtros y Adaptadores

1.2.1. Boquillas

El sensor requiere el uso de una boquilla espirométrica para asegurar un sellado adecuado y evitar fugas durante la inspiración y espiración. Existen dos tipos compatibles:

  • Boquillas espirométricas rígidas
  • Boquillas descartables

1.2.2. Filtro HME

El filtro HME debe colocarse entre el sensor y la boquilla. Su función es proteger el sensor, reducir riesgo de contaminación y mantener resistencia aerodinámica estable.

1.2.3. Clip Nasal

Evita fugas de aire por la nariz y asegura que el flujo pase por el sensor.

1.2.4. Adaptadores 3D

Para pacientes que no logran sellado adecuado con boquillas estándar.

6. Resolución de Problemas

  • No se detecta el sensor → revisar USB y cerrar otros programas.
  • Juego no responde → revisar boquilla, filtro, clip nasal.
  • Rutina no inicia → verificar que existe una asignada.
